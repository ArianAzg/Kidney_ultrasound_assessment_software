Overview 
---------------
This document provides a step-by-step guide of running the VHL study program. Upon successful running of the program, the end-users will be able to see the results of their assessments in an associated CSV sheet to the program. After extraction (for extraction right-click on the zip file and select “Extract All”) of provided zip file under your default download directory (most likely “Downloads” folder in your Windows)
